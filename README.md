# AI价格监控

监控 [链动小铺](https://pay.ldxp.cn) 商品价格变化，自动分类、历史追踪。

## 功能

- **多店铺管理** — 添加/删除店铺，拖动排序
- **自动抓取** — 定时后台刷新商品数据
- **智能分类** — 自动按 Plus/Pro/Team/接码 等分类标记商品
- **价格历史** — 记录每个商品的价格变化，图表展示
- **关键词搜索** — 包含/排除过滤，常用关键词推荐
- **价格区间** — 双滑块筛选指定价格范围
- **主题切换** — 明/暗主题

## 快速开始

需要 Node.js 22 或更高版本。

```bash
# 安装依赖
npm install

# 启动服务
node server.js  # 或双击 start.bat
```

访问 http://localhost:3000

服务默认只监听 `127.0.0.1`。如需局域网或公网访问，请设置 `HOST`，并同时配置高强度 `AUTH_TOKEN`；未配置令牌时服务会拒绝监听非本机地址。可复制 `.env.example` 作为配置参考。

## 技术栈

- **后端** — Node.js + Express + better-sqlite3
- **前端** — 原生 JS + Chart.js（价格图表）
- **抓取** — Axios + Cheerio
